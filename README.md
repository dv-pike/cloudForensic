# Tools:
m365_onedrive_collector.py

# Usage:

Step 1: 

  ```pip install msal```

Step 2: 

  ```m365_onedrive_collector.py "CLIENT_ID" "CLIENT_SECRET" "TENANT_ID" "TARGET_USER"```
  
  e.g.
  
  ```m365_onedrive_collector.py "XXXX" "XXXX" "XXXX" "user1@example.onmicrosoft.com"```
